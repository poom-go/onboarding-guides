전체 진행률 2/7 단계

📋 진행 순서

1

판매 채널 선택
스마트스토어, 쿠팡, 11번가 등 사용 중인 채널을 선택하세요.

2

API 키 발급
각 플랫폼 판매자센터 → 개발자/API 메뉴에서 API 키를 발급받으세요.

3

시스템에 입력
발급받은 API 키를 시스템 판매처 등록 메뉴에 입력하세요.

4

연동 테스트
테스트 버튼을 눌러 정상 연결됐는지 확인하세요.

✅ 완료 체크리스트

💬 자주 묻는 질문

Q. API 키는 어디서 받나요?
각 플랫폼 판매자센터 → 마이페이지 → 개발자 API 메뉴에서 발급받을 수 있어요. 플랫폼마다 위치가 달라서 잘 모르시면 담당자에게 문의해주세요.
Q. 여러 채널을 동시에 등록할 수 있나요?
네, 순서 상관없이 모두 등록 가능해요. 채널별로 각각 API 키를 입력하면 됩니다.
Q. 연동 테스트가 실패해요
API 키를 다시 복사해서 입력해보세요. 공백이 포함되는 경우가 많아요. 그래도 안 되면 담당자에게 문의해주세요.
